Processing p5.js sketches

Playing with processing and p5.js, including exercises from Daniel Shiffman's (excellent) Nature of Code book.

  • Camera: a basic panning camera, originally implemented in Processing, and converted before I realised orbitControl() existed in p5.js. Also allows showing a bounding box to aid orientation, and a starting location.
  • Gaussian Paint Dots (3D): a simple 3D animation that generates 500 randomly coloured spheres, whose locations are normally distributed around origin.
Nature of Code examples and exercises:
  • Exercise 3.1: Angular Motion: a super simple 2D sketch introducing angular rotation in processing.
  • Example 3.2: A Pointing Follower: a simple follower object that accelerates towards the mouse, but also points in the direction it is moving.
  • Exercise 3.2: A Pointing Vehicle: a user controllable "vehicle".
  • Exercise 3.4: Polar Spiral: generating a spiral: an example of where using polar co-ordinates is preferable to cartesian.
